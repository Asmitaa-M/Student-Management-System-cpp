# Student Management System

A console-based Student Management System developed in C++ using file handling and menu-driven operations.

## Features

* Add Student Record
* Display Student Records
* Search Student
* Update Student Details
* Delete Student Record
* Persistent File Storage

## Technologies Used

* C++
* File Handling
* Object-Oriented Programming

## Project Outcome

This application efficiently manages student records with reliable persistent storage and smooth menu-driven navigation.
